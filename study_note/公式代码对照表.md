| 数学符号         | 含义                      | 代码实现（PyTorch / NumPy）                               |
| ------------ | ----------------------- | --------------------------------------------------- |
| xi           | 第 i 个样本的特征              | `X[i]` 或 `X[i, :]`                                  |
| yi           | 第 i 个样本的标签              | `y[i]`                                              |
| wj或 βj       | 第 j 个权重参数               | `model.weight[j]` 或 `theta[j]`                      |
| xiTw         | 线性预测值（点积）               | `torch.dot(X[i], w)` 或 `X[i] @ w`                   |
| ∑i=1n        | 对 i 从 1 到 n 求和          | `torch.sum()` 或 `np.sum()`                          |
| ∏i=1n        | 对 ii 从 1 到 nn 求积        | `torch.prod()` 或 `np.prod()`                        |
| ∥v∥2         | L2 范数（欧几里得距离）           | `torch.norm(v, p=2)`                                |
| ∥v∥22        | L2 范数的平方                | `torch.norm(v, p=2)**2` 或 `(v**2).sum()`            |
| ∥v∥1         | L1 范数（绝对值之和）            | `torch.norm(v, p=1)` 或 `torch.abs(v).sum()`         |
| ∂L/∂w        | 损失函数对 w 的偏导数            | PyTorch 自动：`loss.backward()`，然后 `w.grad`            |
| ∇wL          | 损失函数对 w 的梯度             | `torch.autograd.grad()` 或 `loss.backward()`         |
| 1condition   | 指示函数（条件成立为 1，否则为 0）     | `(condition).float()` 或 `np.where(condition, 1, 0)` |
| max⁡(a,b)    | 取 a 和 b 的最大值            | `max(a, b)` 或 `torch.max(a, b)`                     |
| min⁡(a,b）    | 取 a 和 b 的最小值            | `min(a, b)` 或 `torch.min(a, b)`                     |
| [x]+         | 正部函数（max⁡(0,x)max(0,x)） | `torch.relu(x)` 或 `max(0, x)`                       |
| ⊙            | 逐元素相乘（Hadamard 积）       | `a * b`（PyTorch/NumPy 直接乘）                          |
| ⊗            | 矩阵乘法                    | `A @ B` 或 `torch.mm(A, B)`                          |
| diag(v)      | 以 vv 为对角线的对角矩阵          | `torch.diag(v)`                                     |
| softmax(z)i​ | Softmax 函数              | `torch.softmax(z, dim=-1)`                          |
| σ(x)         | Sigmoid 函数              | `torch.sigmoid(x)`                                  |
| Ex∼p[f(x)]   | 期望值（均值）                 | `torch.mean(f(x))`                                  |
| arg⁡min⁡w​   | 使表达式最小的 ww              | 优化器求解（如 `torch.optim.SGD`）                          |
